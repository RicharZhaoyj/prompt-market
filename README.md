# PromptMarket - AI提示词免费库

## 项目说明

这是一个基于 Next.js 14 的 AI 提示词分享平台。

### 主要功能

- ✨ 32+ 精选AI提示词
- 📂 10个分类：写作、编程、设计、营销、教育、生活、商业、游戏、音乐、其他
- 🔍 分类筛选和搜索
- 📝 用户评价系统
- 📤 提示词提交功能
- 📱 响应式设计

### 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

### 技术栈

- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS
- **语言**: TypeScript
- **部署**: Vercel

### 部署到 Vercel

1. 将代码推送到 GitHub
2. 在 Vercel 中导入项目
3. 自动部署

### 页面结构

- `/` - 首页
- `/prompts` - 提示词列表
- `/prompt/[id]` - 提示词详情
- `/submit` - 提交提示词
- `/blog` - 博客文章
- `/docs` - 帮助文档
